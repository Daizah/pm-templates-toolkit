# PM Templates Toolkit

A collection of project management templates I use across volunteer and professional projects. Built for practical, real-world use, not just theory.

Feel free to use, adapt, or build on any of these for your own projects.

---

## Templates Included

### Planning
- [`project-charter-template.md`](./planning/project-charter-template.md): Defines scope, objectives, stakeholders, and success criteria
- [`kickoff-agenda-template.md`](./planning/kickoff-agenda-template.md): Structured agenda for project kickoff meetings
- [`stakeholder-register-template.md`](./planning/stakeholder-register-template.md): Track stakeholder roles, influence, and communication needs

### Tracking & Execution
- [`raid-log-template.md`](./tracking/raid-log-template.md): Risks, Actions, Issues, and Dependencies tracker
- [`status-report-template.md`](./tracking/status-report-template.md): Weekly project status update for stakeholders
- [`milestone-tracker-template.md`](./tracking/milestone-tracker-template.md): Visual milestone and task tracking

### Closing
- [`retrospective-template.md`](./closing/retrospective-template.md): What went well, what to improve, lessons learned
- [`project-closeout-template.md`](./closing/project-closeout-template.md): Final summary, outcomes, and handoff notes

### Onboarding
- [`volunteer-onboarding-template.md`](./onboarding/volunteer-onboarding-template.md): Onboarding checklist and guide for new team members
